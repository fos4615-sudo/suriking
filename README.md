# 집수리왕

수요자가 집수리 요청과 희망 예산을 등록하고, 여러 작업자가 입찰한 뒤 수요자가 작업자를 선택하여 공사 진행 상태와 대금 지급을 관리하는 로컬 웹앱입니다.
기본 화면에는 실제 서비스 시나리오에 가까운 5건의 테스트 데이터가 포함되어 있습니다.

## 실행 방법

1. 터미널에서 프로젝트 폴더로 이동합니다.
2. 아래 명령으로 로컬 서버를 실행합니다.

```bash
python3 -m http.server 8000
```

3. 크롬에서 `http://localhost:8000` 을 열면 됩니다.

## 주요 기능

- 수요자 수리 요청 등록
- 작업자 입찰 등록 및 최저가 비교
- 수요자 낙찰자 선택
- 요청, 낙찰, 공사중, 공사완료, 입금완료 단계 전환
- 브라우저 `localStorage` 기반 데이터 저장
- 실제형 테스트 데이터 5건 기본 제공

## 반복 테스트

아래 명령으로 5건의 요청 데이터를 기준으로 낙찰, 공사중, 공사완료, 입금완료 흐름을 자동 검증할 수 있습니다.

```bash
python3 test_runner.py
```
