집수리왕

수리요청자가 수리 요청과 희망 예산을 등록하면 여러 공사작업자가 입찰하고, 수리요청자가 직접 낙찰자를 선택한 뒤 공사 진행 상태를 확인하며 대금을 지급하는 플랫폼입니다.

거래 흐름

  1. 수리요청자가 수리 요청과 지급 가능 금액 등록
  2. 공사작업자들이 입찰가와 작업 조건 제안
  3. 수리요청자가 작업자를 선택해 낙찰 확정
  4. 요청, 낙찰, 공사중, 공사완료 단계별 확인
  5. 최종 확인 후 작업자에게 대금 지급 처리
